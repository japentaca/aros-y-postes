# Aros y Postes 3D - Skybox Support

Una aplicación web interactiva de visualización 3D con aros, postes y skybox para entretenerse y relajarse.

## Descripción

Este proyecto es una experiencia visual interactiva en 3D que muestra un paisaje con postes y elementos flotantes en un entorno inmersivo con skybox. La aplicación permite personalizar diversos aspectos del entorno visual para crear una experiencia relajante y entretenida.

## Características

- **Entorno 3D inmersivo**: Utiliza un skybox para crear un entorno visual atractivo
- **Elementos interactivos**: Postes y aros/elementos flotantes en el paisaje 3D
- **Controles de personalización**: Panel de control con múltiples opciones de configuración
- **Monitor de rendimiento**: Contador de FPS (Frames Per Segundo) para monitorear el rendimiento
- **Interfaz responsive**: Diseño adaptable a diferentes tamaños de pantalla
- **Controles intuitivos**: Interfaz de usuario amigable con controles deslizantes

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Three.js (para gráficos 3D)
- WebGL (renderizado 3D)

## Cómo Usar

1. Abre el archivo `index.html` en un navegador web moderno
2. Utiliza los controles en el panel derecho para personalizar el entorno visual
3. Observa el contador de FPS en la esquina superior izquierda para monitorear el rendimiento
4. Disfruta de la visualización 3D relajante y entretenida

## Controles de Personalización

La aplicación incluye un panel de control con las siguientes opciones:
- **Velocidad**: Controla la velocidad de movimiento de los elementos
- **Niebla**: Ajusta la distancia de visibilidad del efecto de niebla
- **Altura de Vuelo**: Modifica la altura de los elementos flotantes
- **Terreno**: Cambia el tamaño del terreno 3D
- **Postes**: Ajusta la cantidad de postes en el paisaje
- **Rivales/Elementos**: Controla la cantidad de elementos flotantes (aros)

## Requisitos del Sistema

- Navegador web moderno con soporte para WebGL
- JavaScript habilitado
- Conexión a internet (para cargar Three.js desde CDN)
